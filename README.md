## Datasets & Files

### student.csv
Contains 35 student records with columns id, name, class, mark, gender.

### world db.sql
Comprehensive SQL queries for the MySQL World Database.

### script examples.sql
Advanced SQL examples including window functions, joins, and more.

### Country Language.sql, country cleaning.sql, city cleaning.sql
SQL scripts for cleaning and standardising tables in the World Database.

### retail_sales_dataset_Master.xlsx
Large 2023 retail dataset with customer and transaction data.

### Bike Sales Datasets
Includes Day_3_Task_1_Bike_Sales_Pivot_Lab.xlsx and Day_3_Task_3_Bike_Sales_Visualisations_Lab.xlsx.
